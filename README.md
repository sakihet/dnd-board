# WIP

DnD board is a web-based, board-style, list-making app. This is a project to create a small Trello clone tailored for minimalists.

![dnd-board](screenshots/dnd-board.png)

## Features

- Create boards
- Create lists
- Create cards
- DnD Board
- DnD List
- Rename
